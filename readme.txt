Плагин правильно склоняет слово "комментарии" в ссылках на комментарии.

== Description ==

Плагин меняет вывод стандартной функции WordPress comments_popup_link, заставляя ее выводить правильные окончания слова &quot;комментарии&quot;: 21 комментарий, 22 комментария, 25 комментариев и т.д.

Помимо этого, с помощью этого плагина можно заменить стандартную функцию comments_popup_link на функцию плагина russian_comments, которая позволит вам выводить произвольный заголовок (title) у ссылок на комментарии.
  
Более подробную информацию вы найдете на [странице плагина](http://www.wordpressplugins.ru/critical/2-4-comment-fix.html).

Если вам понравился мой плагин, то `пожалуйста` поставьте ему 5 звезд.


== Installation ==

1. Скопируйте папку плагина в `/wp-content/plugins/`.
2. Активируйте плагин через меню `Плагины`.
3. Это все.
4. Информацию о замене стандартной функции движка на функцию плагина (которая умеет выводить произвольный заголовок ссылки на комментарии) вы можете найти на [странице плагина](http://www.wordpressplugins.ru/critical/2-4-comment-fix.html).
